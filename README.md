# Text to Speech Converter

A simple Python application that converts text into speech using the `pyttsx3` library. The generated speech can be played directly or saved as an audio file on your local device for later use.

## Features

* Convert text to speech
* Save speech as an audio file
* Offline functionality
* Adjustable voice settings
* Adjustable speech rate
* Lightweight and easy to use
* Cross-platform support

## Requirements

* Python 3.8+
* pyttsx3

## Installation

```bash
git clone https://github.com/ritiksoni5992-hash/Python-text-to-audio.git
cd text-to-speech-converter
pip install pyttsx3
```
